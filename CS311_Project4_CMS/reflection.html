Reflections

  1. I struggled initially with foundations grid layout. It is similar to CSS grid, however,
    it uses a 12 column structure. Foundation had several tutorials to help me learn the grid system,
    making it much easier to use.
  2. I struggled with making the composer works column expand the full width of the page to create the
    desired layout. However, foundation has easy syntax that allowed me to specify that i wanted the column
    to be full height.
  3. After expanding the composer works column to be full height, i then ran into the issue of getting the layout
    correct for the rest of the page. I solved this by making a smaller 8 column grid and placing the rest of the classes
    inside of this grid.
  4. Foundations xy grid system made it fairly easy to get the desired layout, however, I actually think CSS grid layout
    was a little easier to comprehend.
  5. This project was a nice refresher, since I haven't used html and css in a while. It took some research to remember how to
    do certain things, but I got the hang of it.
  6. Another thing I struggled with was aligning text in the center vertically. I solved this by using a flex display and then
    using the aligh-items command to center the text.
  7. Overall, I like Foundations grid layout, however, I would need to spend much more time to figure out the power of it and to
    feel comfortable with it.
